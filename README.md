# F1 · 2025

Race-by-race visualization of the 2025 Formula 1 World Championship, built on top
of the Kaggle [Formula 1 Live Tracker 2024–2026](https://www.kaggle.com/datasets/devtayyabsajjad/formula-1-live-tracker-2024-2026-dataset)
dataset.

## Views

1. **Season** (`/`) — index of all 24 rounds with winners. *(Built — basic table.)*
2. **Race** (`/race`) — animated grid → finish on a stylized track SVG, with pit stops marked at their lap. *(Coming.)*
3. **Standings** (`/standings`) — championship points race over the season. *(Coming.)*

## Stack

- Next.js 16 (App Router) · TypeScript · Tailwind v4 · shadcn/ui
- `better-sqlite3` for the local DB (file at `data/f1.db`)

## Data

CSVs live in `data/raw/` and are ingested into `data/f1.db`:

```
circuits.csv             constructor_standings.csv  constructors.csv
data_dictionary.csv      driver_standings.csv       drivers.csv
pit_stops.csv            qualifying.csv             race_results.csv
races.csv
```

This is **race-summary** data (final position, grid, laps, pit stops, qualifying
times). It's not lap-by-lap and not telemetry, so the race view will animate
grid → finish rather than a true racing line.

### Build

```bash
npm install
npm run ingest   # reads data/raw/*.csv, writes data/f1.db
npm run dev      # http://localhost:3000
```
